# 文章转图技能

将文章、笔记或文字内容转化为**实验室精密手册感 + 波普实验风格**的小红书高密度信息图。

## 工作流程

1. **内容分析** — 提取核心主题和关键信息点
2. **信息提炼** — 筛选 6-7 个核心模块（实用性、稀缺性、清晰性）
3. **视觉坐标建立** — 为每个模块分配坐标标签（A-01、B-05...）
4. **生成高密度内容** — 填充品牌名、数值、百分比及视觉符号
5. **生图** — 调用 API 生成信息图，自动推送飞书
