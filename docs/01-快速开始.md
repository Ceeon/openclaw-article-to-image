# 快速开始

## 获取 API Key

本技能需要调用 AI 生图 API。如果你还没有 API Key，请先注册：

| 平台 | 地址 | 说明 |
|------|------|------|
| **yunwu.ai** | https://yunwu.ai | 注册后在控制台生成 Key，支持多种主流生图模型 |
| **apimart.ai** | https://apimart.ai | 注册后在 API Keys 页面创建密钥，本技能默认使用此平台 |

## 配置 .env

```bash
cd ~/.openclaw/skills/article-to-image
cp .env.example .env
vim .env
```

`.env.example` 内容：

```
# 生图 API（必填）
API_KEY=your-api-key-here

# 飞书推送（选填，不配置则跳过推送）
FEISHU_APP_ID=your-app-id
FEISHU_APP_SECRET=your-app-secret
FEISHU_USER_ID=your-user-id
```

## 运行

```bash
~/.openclaw/skills/article-to-image/scripts/generate_image.sh "<prompt>" [resolution]
```

- **resolution**：`1K`（默认）/ `2K` / `4K`
- **输出目录**：`~/projects/todohub/generated_images/`
- 配置了飞书密钥后，生成完会自动推送到飞书
